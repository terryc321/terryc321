What is Clojurescript ?

clojurescript is a language that translates to javascript

clojurescript has interop with javascript. can call javascript code from clojurescript, call clojurescript code from javascript

What Advantages ? Why not just plain Javascript ?

Javascript is a flawed language. There is good parts, but akin to walking through a field with land mines.

Clojurescript compiles down to javascript using Google Closure Compiler. The resultant javascript code is jibberish and unreadable. It becomes rewritten and minified for production use. There may be source tracking for development work but I could not make sense of where it was. javascript console just goes nuts.

Clojurescript applications must be built incrementally , there is a slight disconnect between writing the application and seeing the results. Sometimes if there is an error but there is no output and it is hard to diagnose where the mistake has been made in the code. Although figwheel:main does now provide an excellent console REPL. As long as developer uses figwheel quick code - test cycle - takes about 10 seconds for results to become available on the browser, there is no problems I can see.

Clojurescript can use the google closure library , which is huge. I think i saw mention of if code tampers with prototype object then it can really screw up the closure library code.

Have yet to see how clojurescript integrates into an application with several big javascript libraries.
